Đối chiếu cấu trúc bảng — Excel ↔ UI hiện tại ↔ Đề xuất

Cấu trúc header (rowSpan/colSpan) lấy trực tiếp từ file SOP Excel qua phân tích merged cells. Mỗi bảng kèm 3 phần: (1) Excel gốc — (2) UI hiện tại — (3) Đề xuất align với Excel. Field mapping với code dưới mỗi bảng.

1. TabDoanhThu 2. TabSanLuong 3. ChiTieuClient 4. BaoCaoSL 5. BaoCaoDT 6. MonthlyReport 7. DebtMatrix

Tóm tắt tình trạng

  1. TabDoanhThu ↔ Excel BC_DoanhThu rows 8-9 — có 4 nhóm cần colSpan: Doanh thu Thô / Trát / Tổng / Tỷ lệ%
  2. TabSanLuong ↔ Excel BC_SanLuong rows 7-8 — có 2 nhóm: Sản lượng thực hiện thô / Tỷ lệ%
  3. ChiTieuClient ↔ Excel CT_SL_DT rows 8-9 — có 2 nhóm: Sản lượng kỳ này / Doanh thu kỳ này, mỗi nhóm [Chỉ tiêu | Thực hiện]
  4. BaoCaoSL ↔ Cùng cấu trúc BC_SanLuong (ở chế độ xem-only)
  5. BaoCaoDT ↔ Cùng cấu trúc BC_DoanhThu (ở chế độ xem-only)
  6. MonthlyReport ↔ Excel Báo Cáo Tháng rows 7-8 — 2 nhóm lớn: THỰC TẾ / HỢP ĐỒNG, mỗi nhóm 4 cột [Đầu kỳ | PS Phải Trả | PS Đã Trả | Cuối kỳ]
  7. DebtMatrix ↔ Excel Tổng Hợp Công Nợ / Công Nợ — đã làm 3-tier đúng pattern Excel ✓
