Cải thiện chính:
  • Header 2 tầng: Tier 1 = nhóm (Thô · Trát · Tổng DT · %), Tier 2 = chi tiết (Kỳ · Lũy kế · Còn lại). Bỏ ký hiệu (D)–(Q).
  • Background nhạt theo nhóm giúp mắt scan ngang nhanh, tách biệt rõ vùng dữ liệu.
  • Border-l 2px giữa các nhóm = đường ngăn dày hơn → giống bảng Excel.
  • Tabular-nums + monospace cho cột số → hàng nghìn/triệu thẳng cột, không "nhảy" khi giá trị thay đổi.
  • Phân biệt input vs computed: ô nhập (vàng nhạt) vs ô tự tính (xám) → user biết chỗ nào click được.
  • Sticky lô cố định bên trái khi scroll ngang.
  • Spacing thoáng hơn: padding 6px/10px thay vì 1px/2px.

Ghi chú kỹ thuật

Mapping cột với code hiện tại:

  • HĐ (D) → contractValue · KH kỳ (E) → dtKeHoachKy
  • Thô [Kỳ/LK/Còn] = dtThoKy / dtThoLuyKe / contractValue − dtThoLuyKe
  • Trát [QT/Kỳ/LK/Còn] = qtTratChua / dtTratKy / dtTratLuyKe / qtTratChua − dtTratLuyKe
  • Tổng [Kỳ/LK/Còn] = F+J / G+K / H+L
  • % [vs KH/vs HĐ] = dtThoKy/dtKeHoachKy · dtThoLuyKe/contractValue

Pattern này có thể tái sử dụng cho TabSanLuong, BaoCaoSL, BaoCaoDT, MonthlyReport — tách thành component <GroupedTableHeader groups={[...]}/>.
