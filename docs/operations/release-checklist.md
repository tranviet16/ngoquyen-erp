# Release checklist

- [ ] `npm run risk:verify`, unit, integration and E2E-security suites are green.
- [ ] Migration rehearsal and disposable restore drill have current evidence.
- [ ] P0 permission smoke covers auth, ACL, export, payment, import and attachments.
- [ ] Dependency/secret findings are triaged with owner and deadline.
- [ ] Rollout, rollback and data-repair owners are assigned.
- [ ] Performance warnings in scope are `cleared` with re-measurement evidence.
