# Quality loop

Daily: triage P0/P1 alerts, audit failures, backup misses, denial spikes and payment/import errors. Weekly: review incidents, flaky expiry, query deltas and unresolved findings; select at most three evidence-backed improvements. Release: complete the release checklist. Monthly: review P0 threat model, manifest drift and kill switches.

An incident closes only with owner, root cause, regression test and recovered verification metric.
